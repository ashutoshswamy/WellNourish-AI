# **App Name**: WellNourish AI

## Core Features:

- Profile Creation: Users input personal data (age, gender, height, weight, activity level) for personalized planning.
- Goal Setting: Users specify their fitness goals (e.g., weight loss, muscle gain).
- AI-Powered Diet Plan Generation: AI generates a customized diet plan tool, based on user data and goals.
- AI-Powered Workout Regimen Generation: AI generates a personalized workout regimen tool, tailored to the user's profile and objectives.
- Plan Display: Display diet plan and workout regimen in a clear, structured format.
- Progress Tracking: Allow users to track their progress and visualize achievements.
- Guidance: Basic educational information regarding exercises

## Style Guidelines:

- Primary color: Forest green (#33704B), evoking nature, health, and growth.
- Background color: Pale green (#E0E5E2), a light and unobtrusive background.
- Accent color: Warm Yellow (#E7B751), highlighting important information.
- Body and headline font: 'PT Sans', a humanist sans-serif suitable for both headlines and body text, providing a modern and friendly feel.
- Use clean, modern icons to represent exercises, food groups, and progress metrics.
- Ensure a clean and intuitive layout for easy navigation and data presentation.
- Use subtle animations to provide feedback and guide the user through the app.
