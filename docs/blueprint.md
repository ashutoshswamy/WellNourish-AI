# **App Name**: WellNourish AI

## Core Features:

- Health Profile Creation: Allow users to create a detailed health profile, including medical history, lifestyle, and fitness goals.
- Personalized Diet Plans: Use AI to analyze the user's health profile and preferences to generate personalized diet plans.
- Custom Workout Plans: Employ an AI tool to develop custom workout plans based on the user's fitness level, goals, and available equipment.
- Plan Display: Enable users to view detailed diet and workout plans.
- Plan Saving: Offer users the option to save generated plans for future access.
- Progress Tracking: Allow users to track their progress against set goals, updating and visualizing the current plan
- Supabase Authentication: Use Supabase for authentication using Supabase npm packages

## Style Guidelines:

- Primary color: Deep sky blue (#3EB489), providing a sense of trust and tranquility.
- Background color: Light beige (#F5F5DC), providing a calm backdrop
- Accent color: Mellow apricot (#F29987), which suggests liveliness and activity, creating contrast to enhance UX.
- Font pairing: 'Belleza' (sans-serif) for headlines, with its artful, stylish touch; and 'Alegreya' (serif) for body text, adding readability for longer text.
- Use clean, modern icons to represent different exercises and dietary elements.
- A clean, user-friendly layout with intuitive navigation for ease of use.
- Subtle animations for progress tracking and goal achievements.
