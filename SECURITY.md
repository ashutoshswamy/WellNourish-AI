# Security Policy

## Reporting Security Vulnerabilities

WellNourish AI takes security seriously. If you discover a security vulnerability, please send an email to security@wellnourishai.in with details of the vulnerability.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Security Features

- HTTPS encryption for all communications
- Secure authentication with Supabase
- Data encryption at rest and in transit
- Regular security updates and patches
- GDPR and privacy compliance

For more information about our security practices, please visit our [Privacy Policy](/privacy).
