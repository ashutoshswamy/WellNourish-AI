# WellNourish AI

WellNourish AI is a fast, AI-powered health and fitness application designed to help users achieve their wellness goals through personalized diet and workout plans. Built with the latest web technologies, it offers a seamless and engaging user experience.

## Features

- **Personalized Plans**: Generates 7-day or 30-day diet and workout plans tailored to your health profile and goals using Google's Gemini AI.
- **Profile Management**: comprehensive user profiles tracking age, weight, height, activity level, medical conditions, allergies, and dietary preferences.
- **Smart Authentication**: Secure login via Google and GitHub using Supabase Auth.
- **Dynamic Dashboard**: Interactive dashboard to view and manage your generated plans.
- **Modern UI/UX**: Aesthetic design with smooth animations using Framer Motion and a responsive layout.
- **Multiple Plans**: Save and retrieve multiple generated plans.

## Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Database & Auth**: [Supabase](https://supabase.com/)
- **AI Model**: [Google Gemini](https://deepmind.google/technologies/gemini/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Form Handling**: React Hook Form + Zod
- **Email**: Resend (planned/integrated)
