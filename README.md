## 🌟 Features

### 🔐 Authentication & User Management

- **Secure User Registration**: Email-based signup with password confirmation
- **User Authentication**: Secure login/logout with Supabase Auth
- **Protected Routes**: Authentication-required access to personal features
- **Password Security**: Minimum 6-character password requirements

### 👤 Personalized Profile System

- **Comprehensive Profile Creation**: Input personal data including:
  - Age, gender, height (cm/ft-inches), weight (kg/lbs)
  - Activity level (sedentary to extra active)
  - Fitness goals (weight loss, muscle gain, maintenance, etc.)
- **Advanced Dietary Preferences**: Support for multiple dietary styles:
  - Vegetarian, Vegan, Keto, Paleo, Mediterranean, Low-carb, etc.
  - Preferred cuisines (Italian, Indian, Chinese, Mediterranean, etc.)
  - Allergy and intolerance management (nuts, gluten, dairy, etc.)
- **Profile Management**: Update and modify your profile anytime

### 🤖 AI-Powered Plan Generation

- **Smart Diet Plans**: AI-generated personalized meal plans based on:
  - Your physical profile and goals
  - Dietary preferences and restrictions
  - Preferred cuisine types
  - Complete allergen avoidance
- **Custom Workout Regimens**: Tailored exercise routines featuring:
  - Weekly schedule breakdown
  - Specific exercises with sets and reps
  - Rest days and recovery guidance
  - Progression tips and safety considerations
- **Instant Generation**: Get complete wellness plans in seconds

### 📊 Plan Management & Organization

- **Personal Dashboard**: Overview of your wellness journey with:
  - Total plans created
  - Profile completion status
  - Recent plan history
- **Plan Library**: Comprehensive plan management:
  - View all your generated plans
  - Search and filter functionality
  - Plan deletion and organization
- **Detailed Plan Views**:
  - Tabbed interface for diet and workout plans
  - Full-screen plan viewing
  - Markdown-formatted, easy-to-read content

### 🎨 User Experience

- **Responsive Design**: Optimized for all devices (mobile, tablet, desktop)
- **Modern UI**: Clean, accessible interface with Tailwind CSS and Radix UI
- **Loading States**: Smooth user experience with proper loading indicators
- **Error Handling**: Comprehensive error messages and validation
- **Progress Tracking**: Visual indicators of plan generation progress
